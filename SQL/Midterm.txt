5. Referential integrity helps us validate the contents of a relational database. When updating a value inside a table, we must make sure that any foreign keys in that row exist as a primary key in another table. Additionally, when deleting an entry, we must make sure that the primary key of that entry does not exist as a foreign key in another table.

6. SELECT A.ENAME AS "Manager's Name", B.DNAME AS "Manager's Department name", C.EMPNO AS "Employee Number", C.ENAME AS "Employee Name"
FROM EMP A, DEPT B, EMP C
WHERE A.DEPTNO = B.DEPTNO AND A.EMPNO = C.MGR;
